
# Fitness App


## Technologies and Libraries

- React Js
- Tailwind CSS
- React Icons
- Firebase (Google, Twitter, Github Login, Register and Login)

  
## Installation 

```bash 
    git clone https://github.com/mustafakaracuha/react-fitness-app.git
```
```bash 
    cd react-fitness-app
```
```bash 
    npm install react-fitness-app
```
```bash 
    npm run dev
```
